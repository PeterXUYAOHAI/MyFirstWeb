Style:
For the first page, I use fixed layout to avoid unexpected change.I put the banner on the top,the video on the left-bottom and the navigation and ifame on the right-bottom
I use iframe to show the multipages. And mostly I use external CSS. I also add some CSS animation to banner and navigation.
I also set alt for all the images.
To make sure the text can show properly, I used default font.


Banner:
Banner is rotated and color will chang when hover on it, the subtitle has animation and linked to this page.


Video:
The video will be played automaticly and over and over again, the user can control it.


Navigation:
The navigation will change color and rotate when hover on it.


Plot:
I used column count to divide large number of words to make it easier to read.


Cast:
I put the image and text in different sequence for different actors. The cursor will also be invisible when in this page.
The subtitle used "clear: both".


Award:
I put a image as background. The position of title and table are justified for the image. The background color of cells will be change when hover on it.


Gallary:
I used skew for the title. The images will be enlarged when mouse hover on it.
